# CHANGELOG

## v1.3.0 (2026-06-24)

- 支持 rua 其他群员：@另一位群员 + `rua他`/`rua她`/`rua它`/`rua一下`/`rua`（全文），摸摸被 @ 的人

## v1.2.0 (2026-06-24)

- 支持私聊触发：私聊中无需 @，直接发送指令即可 rua

## v1.1.0 (2026-06-24)

- 精确匹配：仅 `rua`（全文）/ `rua我` / `rua一下` 触发 rua 发送者
- 新增 `rua你` / `rua自己` 指令，机器人摸自己头像
- 新增频率限制：同 QQ 号 2 分钟内仅允许成功 1 次，超频提示

## v1.0.0 (2026-06-24)

- 初始版本
- 支持 @机器人 + `rua` 触发摸头GIF生成
- 基于 UAPI `motou` API 生成 Petpet 风格动图
